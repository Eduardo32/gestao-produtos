Em que consiste o projeto

Um CRUD simples onde é possivel:

Criar uma conta Logar Registrar Produtos
Você pode criar uma conta Username e Senha que você usara para se autenticar na aplicação. Para criar uma conta click aqui.
Depois que você criar uma conta ou caso você já tenha uma pode fazer login aqui.
E uma vez dentro da aplicação você podera cadastrar produtos com nome, descrição, quantidade e valor. Podera tambem editar as produtos cadastrados clicando no botão editar na listagem de produtos ou se desejar pode tambem excluir os produtos.
